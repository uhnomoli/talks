What is test-driven development?

Vocab

There are different types of testing

Unit testing

(smallest testable part of an application)

Integration testing

(interaction between components)

System testing

(testing the system as a whole)

TDD is largely focused on unit testing

Harness

(the framework for writing tests)

Fixture

(means of configuration enabling the repeatability of tests)

Case

(a single test)

TDD cycle

  • Red Enforces a specification
  • Green Implementation meets the specification
  • Refactor Improve the implementation

Recapping

  • Types of testing: unit, integration, system
  • Unit tests consist of: harness, fixtures, cases
  • Development cycle of: red, green, refactor

cmake_minimum_required(VERSION 3.11.2)

project(dictionary)

add_subdirectory(src)
add_subdirectory(vendor)

include(CTest)
if(BUILD_TESTING)
    add_subdirectory(test)
endif()

add_library(dictionary SHARED dictionary.c)
target_compile_options(
    dictionary PUBLIC
        -Waggregate-return -Wall -Wextra -Wfloat-equal -Winline
        -Wpedantic -Wstack-usage=1024 -Wwrite-strings)
target_include_directories(dictionary PUBLIC ${PROJECT_SOURCE_DIR}/src)


add_executable(test_dictionary test_dictionary.c)
target_link_libraries(test_dictionary cmock)
add_test(test_dictionary test_dictionary)

add_library(unity STATIC unity/unity.c)
target_include_directories(unity PUBLIC ${PROJECT_SOURCE_DIR}/vendor/unity)

add_library(cmock STATIC cmock/cmock.c)
target_include_directories(cmock PUBLIC ${PROJECT_SOURCE_DIR}/vendor/cmock)
target_link_libraries(cmock unity)

Algorithm

(recipe for solving a problem)

In our case, interpret data unassisted (mostly)

Two ways to group machine learning algorithms

How they learn

(how facts are determined from input)

How they function

(how decisions are made based on those facts)

Ways they learn

(there are a few)

Supervised

(initial set of data is labeled)

Unsupervised

(data is unlabeled and facts are deduced from patterns)

Reinforcement

(data is unlabeled and decisions are made based on rewards)

Ways they function

(there are a lot)

Clustering

(organizing input based on observed patterns)

Instanced-based

(compares to previous output)

Artificial neural networks

(simulates neurons in the brain)

Recapping

  • Algorithms are the core of how machines learn
  • They are categorized by how they learn and function
  • They operate on features and/or labels
