Algorithm

(recipe for solving a problem)

In our case, interpret data unassisted (mostly)

Two ways to group machine learning algorithms

How they learn

(how facts are determined from input)

How they function

(how decisions are made based on those facts)

Ways they learn

(there are a few)

Supervised

(initial set of data is labeled)

Unsupervised

(data is unlabeled and facts are deduced from patterns)

Reinforcement

(data is unlabeled and decisions are made based on rewards)

Ways they function

(there are a lot)

Clustering

(organizing input based on observed patterns)

Instanced-based

(compares to previous output)

Artificial neural networks

(simulates neurons in the brain)

Recapping

  • Algorithms are the core of how machines learn
  • They are categorized by how they learn and function
  • They operate on features and/or labels
